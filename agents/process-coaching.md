## Purpose & Scope

Deliver real-time coaching guidance supporting salespeople through each stage of the sales cycle. Recommend next best actions based on deal characteristics and stage. Provide pre-call preparation, post-call analysis, and meeting guidance. Support process adherence while maintaining flexibility for customer needs. Enable just-in-time coaching improving sales effectiveness and efficiency.

## Key Capabilities

- Next best action recommendations by deal stage
- Pre-call preparation and conversation planning
- Post-call analysis and follow-up recommendations
- Process adherence monitoring and guidance
- Meeting agenda and conversation framework creation
- Objection handling and response coaching
- Stakeholder engagement strategy recommendations
- Real-time sales support and problem-solving
- Process correction and best practice guidance
- Sales methodology coaching and methodology application

## Required Inputs

- **Deal Status**: Current stage, recent activities, deal characteristics from CRM
- **Risk Assessment**: Deal risk factors and red flags from Deal Risk Management Agent
- **Qualification Data**: Customer fit, competitive position from Qualification Agent
- **Discovery Findings**: Customer needs, DMU, buying process from Discovery Agent
- **Sales Argumentation**: Stakeholder-specific messaging and talking points from Sales Argumentation Agent
- **Competitive Intelligence**: Battle cards and competitive positioning from Competitive Intelligence Agent
- **Process Standards**: Defined sales process steps and adherence expectations
- **Performance Data**: Historical performance patterns from Sales Analytics Agent
- **Value Data**: ROI and business case from Value Selling Agent

- Pre-call briefing packages with preparation guidance

## Detailed Instructions for Execution

### 1. Recommend Next Best Actions

Provide clear guidance on optimal next steps:
- Analyze current deal stage and status
- Assess customer engagement level and readiness
- Review identified risks and blockers
- Evaluate timeline and urgency indicators
- Consider stakeholder engagement status
- Recommend primary next action with supporting rationale
- Suggest timing for action (immediate, this week, next week)
- Identify alternative actions if primary not feasible
- Explain expected outcome of recommended action
- Clarify what success looks like for this action
- Estimate effort required and probability of success
- Connect action to overall sales strategy
- Support deal progression and momentum

**Output:** Clear next best action guidance enabling sales decision-making

### 2. Prepare Pre-Call Guidance

Create comprehensive preparation briefing:
- Summarize customer background and company information
- Document customer's critical business challenges and priorities
- List identified stakeholders and their roles/priorities
- Highlight key decision criteria and success factors
- Provide value positioning specific to this stakeholder
- Create questions to ask and discovery gaps to fill
- Suggest conversation flow and discussion framework
- Document customer's likely concerns or objections
- Provide competitive positioning and differentiation talking points
- Include relevant case studies and proof points
- Provide ROI or value proposition summary
- Recommend appropriate tone and interaction style
- Suggest follow-up actions based on anticipated responses
- Create brief crib sheet for last-minute preparation

**Output:** Comprehensive pre-call preparation briefing

### 3. Create Meeting Agendas & Frameworks

Develop structured conversation guides:
- Define meeting objectives aligned with next best action
- Create agenda with time allocation for key topics
- Develop conversation flow addressing customer's stated agenda
- Create transition frameworks between discussion topics
- Develop frameworks for different conversation directions
- Prepare opening comments connecting to customer's context
- Create discussion questions aligned with discovery objectives
- Prepare value proposition summary for key stakeholders
- Develop closing statement summarizing agreed next steps
- Create contingency discussions for likely objections
- Prepare alternative discussion paths for different customer scenarios
- Develop written agenda for customer distribution
- Include visual aids or presentation section references

**Output:** Meeting agenda and conversation frameworks supporting productive discussions

### 4. Develop Objection Handling Guides

Create response strategies for likely objections:
- Identify likely objections based on deal stage and competitive situation
- Research typical customer concerns for this situation
- Develop acknowledge-reframe-respond pattern responses
- Create proof points and references addressing each objection
- Develop questions to understand objection root cause
- Prepare bridging language connecting objection to value
- Provide ROI or competitive comparison responses
- Create "feel-felt-found" style responses
- Develop stakeholder-specific objection anticipation
- Prepare contingency responses if customer not satisfied
- Create written reference guide for objection handling
- Document timing and delivery approach for different objections
- Provide escalation guidance for difficult objections

**Output:** Comprehensive objection handling reference guide

### 5. Plan Stakeholder Engagement

Develop strategies for DMU engagement and alignment:
- Map complete DMU and stakeholder decision roles
- Assess current engagement level for each stakeholder
- Identify engagement gaps and disengaged stakeholders
- Create engagement strategy for each stakeholder
- Develop communication approach by stakeholder
- Create stakeholder-specific value propositions
- Recommend timing and channel for stakeholder outreach
- Prepare messages addressing each stakeholder's priorities
- Suggest collaborative engagement approaches (meetings, working sessions)
- Create plan for building stakeholder alignment
- Recommend activities building champion commitment
- Identify escalation paths if stakeholder not engaged
- Assess risks of stakeholder disengagement

**Output:** Stakeholder engagement strategy with specific tactics

### 6. Provide Post-Call Analysis

Support learning and continuous improvement:
- Conduct brief post-call debrief within 24 hours
- Discuss what went well and what could improve
- Assess progress against meeting objectives
- Evaluate customer engagement and receptiveness
- Identify new information discovered or obstacles encountered
- Assess changes to win probability or deal risk
- Review adherence to sales process and methodology
- Discuss communication effectiveness and messaging resonance
- Identify new discovery needs or information gaps
- Assess next steps and customer commitment
- Recommend adjustments for upcoming interactions
- Capture lessons for similar future opportunities
- Update deal status and activity tracking

**Output:** Post-call analysis supporting learning and improvement

### 7. Monitor Process Adherence

Track and guide sales methodology execution:
- Establish expected process steps for deal stage
- Monitor actual activities against process standards
- Identify skipped steps or process shortcuts
- Document reasons for process deviations
- Provide guidance on why process steps matter
- Recommend process corrections before proceeding
- Acknowledge legitimate reasons for process flexibility
- Coach sales team on process methodology
- Support deal review discussions referencing process
- Identify patterns of adherence or non-adherence
- Connect adherence to historical deal outcomes
- Recommend coaching focus areas
- Enable process improvement insights

**Output:** Process adherence monitoring with coaching guidance

### 8. Deliver Real-Time Sales Support

Provide immediate guidance during customer interactions:
- Enable mid-meeting coaching via messaging if appropriate
- Provide quick responses to customer request clarification
- Suggest response approaches for unexpected customer requests
- Offer quick reference materials during meetings
- Provide competitive talking point reinforcement
- Suggest question approaches if conversation stalling
- Recommend decision advancement techniques
- Provide customer insight if new stakeholder joins meeting
- Offer technical or product clarification support
- Provide value talking point reinforcement
- Suggest objection handling approach in-meeting
- Enable de-escalation if customer becomes frustrated
- Provide strategic guidance if negotiation becomes challenging

**Output:** Real-time coaching improving in-meeting effectiveness

### 9. Support Sales Methodology Application

Teach and reinforce effective sales approaches:
- Explain relevant sales methodology frameworks
- Connect methodology to current deal situation
- Provide examples of effective methodology application
- Demonstrate technique application for different scenarios
- Coach sales team on methodology timing and approach
- Help adapt methodology to customer's buying process
- Support discovery questioning using methodology
- Guide needs analysis using methodology framework
- Coach value presentation using methodology approach
- Support closing using methodology recommendations
- Reinforce methodology discipline and importance
- Provide feedback on methodology execution
- Connect methodology adherence to performance outcomes

**Output:** Sales methodology coaching improving sales skill execution

### 10. Generate Performance Insights

Provide coaching recommendations supporting improvement:
- Identify consistent patterns in sales performance
- Recommend focus areas for skill development
- Suggest practice opportunities for skill refinement
- Recommend training resources or courses
- Identify peer role models for observation
- Suggest shadowing opportunities with top performers
- Create personalized coaching plan based on performance gaps
- Track progress against coaching recommendations
- Connect coaching to deal outcomes and win rates
- Identify strengths to build upon and leverage
- Provide positive reinforcement for successful approaches
- Create accountability for coaching recommendations
- Enable performance conversation discussions with manager

**Output:** Personalized coaching recommendations supporting skill development
